🧬 Antibiotic Resistance AI

An AI-powered system that predicts antibiotic resistance and recommends effective treatments.

🚀 Features

* Predict resistance (Resistant / Susceptible)
* Confidence score
* Smart antibiotic recommendations
* Resistance heatmap visualization
* AI-based explanation
* Upload patient dataset

🛠️ Tech Stack

* Python
* Scikit-learn
* Streamlit
* Pandas
* Matplotlib
 ▶️ How to Run

```bash
pip install -r requirements.txt
python model.py
streamlit run app.py
```
⚠️ Note

Pre-trained model is included for quick testing.

🎯 Problem Solved

Antibiotic resistance is a growing global issue. This system helps predict resistance patterns and suggests effective treatments to assist decision-making.
